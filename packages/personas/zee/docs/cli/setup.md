# `zee setup`

Initialize `~/.zee/zee.json` and the agent workspace.

Related:
- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
zee setup
zee setup --workspace ~/zee
```

To run the wizard via setup:

```bash
zee setup --wizard
```
