Agent-Core can run inside GitLab pipelines or Duo workflows. This page is intentionally short while the integration story is refined.

## GitLab CI

- Run Agent-Core in CI jobs on your runners.
- Keep configuration in `.agent-core/agent-core.jsonc`.
- Store secrets in GitLab CI variables.

## GitLab Duo

- Mention `@agent-core` in comments to trigger tasks (when configured).
- Follow GitLab's Duo Agent Assistant setup guidance for runners and permissions.
