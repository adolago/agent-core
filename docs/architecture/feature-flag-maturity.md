Source: `packages/agent-core/src/flag/flag.ts`.

| Flag | Purpose | Maturity | Owner | Target milestone |
| --- | --- | --- | --- | --- |
| `OPENCODE_EXPERIMENTAL` | Global toggle for experimental features | Experimental | Core | M0 |
| `OPENCODE_ENABLE_EXPERIMENTAL_MODELS` | Allow experimental model definitions | Experimental | Core | M1 |
| `OPENCODE_EXPERIMENTAL_FILEWATCHER` | Enable file watcher | Experimental | Core | M1 |
| `OPENCODE_EXPERIMENTAL_DISABLE_FILEWATCHER` | Disable file watcher | Experimental | Core | M1 |
| `OPENCODE_EXPERIMENTAL_ICON_DISCOVERY` | Enable icon discovery in UI | Experimental | CLI | M1 |
| `OPENCODE_EXPERIMENTAL_DISABLE_COPY_ON_SELECT` | Disable copy on select | Experimental | CLI | M1 |
| `OPENCODE_ENABLE_EXA` | Enable Exa search integration | Experimental | Core | M2 |
| `OPENCODE_EXPERIMENTAL_BASH_MAX_OUTPUT_LENGTH` | Limit bash output length | Experimental | Core | M1 |
| `OPENCODE_EXPERIMENTAL_BASH_DEFAULT_TIMEOUT_MS` | Default bash timeout | Experimental | Core | M1 |
| `OPENCODE_EXPERIMENTAL_LLM_STREAM_START_TIMEOUT_MS` | Stream start timeout | Experimental | Core | M1 |
| `OPENCODE_EXPERIMENTAL_OUTPUT_TOKEN_MAX` | Max output tokens | Experimental | Core | M1 |
| `OPENCODE_EXPERIMENTAL_OXFMT` | Enable oxfmt integration | Experimental | CLI | M2 |
| `OPENCODE_EXPERIMENTAL_LSP_TY` | Enable ty LSP server | Experimental | LSP | M2 |
| `OPENCODE_EXPERIMENTAL_LSP_TOOL` | Enable LSP tooling | Experimental | LSP | M2 |
| `OPENCODE_EXPERIMENTAL_PLAN_MODE` | Enable plan mode | Experimental | CLI | M1 |
