## How to Hide Models/Providers

### Method 1: Disable Entire Providers

Edit `~/.config/agent-core/agent-core.jsonc`:

```json
{
  "disabled_providers": ["openai", "google", "xai"]
}
```

### Method 2: Blacklist Specific Models

```json
{
  "provider": {
    "anthropic": {
      "blacklist": ["claude-opus-4-5"]
    },
    "openai": {
      "blacklist": ["gpt-4o"]
    }
  }
}
```

### Method 3: Whitelist Only Specific Models

```json
{
  "provider": {
    "anthropic": {
      "whitelist": ["claude-sonnet-4-5", "claude-haiku-4-5"]
    }
  }
}
```

### Method 4: Zee Gateway Config

Edit `~/.zee/config.json`:

```json
{
  "agents": {
    "defaults": {
      "models": {
        "anthropic/claude-sonnet-4-5": { "enabled": false }
      }
    }
  }
}
```
